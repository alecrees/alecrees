# Alec Rees

Product developer and creative technologist building native mobile, web and music-technology products. I am the founder of [Metopida Ltd](https://metopida.com/) and the product developer behind [Metopus](https://metopus.com/).

My work spans product strategy, UX and original visual design, implementation, integration, documentation and practical testing. I currently work across Android/Kotlin, iOS/Swift, React/TypeScript, Rust/WebAssembly and Supabase/PostgreSQL.

## Featured Project

### Metopus

[Metopus](https://metopus.com/) is an artist and fan platform developed through Metopida Ltd. It grew from roughly two decades of first-hand music-industry experience and three years of product research, prototyping and development.

The production repositories remain private because they contain commercial product logic, backend and security work, unpublished assets and protected implementation. The public case study provides reviewable technical evidence without exposing that material.

- **[Product and engineering case study](https://github.com/alecrees/metopus-product-case-study):** architecture, screenshots, implementation decisions, testing strategy and confidentiality boundaries
- **[Interactive Culture/Scene demo](https://alecrees.github.io/metopus-product-case-study/):** React/TypeScript interface with a public-safe Rust/WebAssembly layout engine
- **[Portfolio case study](https://alecreesdrummer.com/#metopus):** product overview, original visual work and selected application screens
